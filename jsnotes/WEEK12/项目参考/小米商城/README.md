# mishop-react(小米商城)
## 主要技术栈
- React
- React-Router
- React-Redux
- node.js
- express
- less
## 项目运行步骤
- 1、把项目克隆到本地
> - https://github.com/jeechug/mishop-react.git

- 2、进入/切换到项目根目录并安装依赖
> - cd mishop-react
> - yarn install / npm install

- 3、运行server服务
> - server文件右键=>run 'server.js'

- 4、启动项目
> - yarn start / npm start
